# Dhun — Product Design System

## Design Philosophy

- Loud but controlled
- Not minimal
- Not messy
- Emotional + expressive

---

## Layout System

- Strong spacing scale (8pt grid)
- Large padding
- Clear hierarchy

---

## Typography

- No Inter
- Use:
  - Neue Montreal / Clash Display
  - Clean readable body font

---

## Colors

Base:
- #0F0F0F
- #F5F5F5

Accents:
- Neon purple
- Electric blue
- Hot pink
- Acid green

---

## Components

### Cards
- Rounded (24–32px)
- Mixed styles (glass + solid)
- Shadow depth

---

### Buttons
- Thick
- Rounded
- Gradient
- Hover inflate
- Click compress

---

### Inputs
- Large
- Friendly
- Placeholder expressive

---

## Dedication Screen

Layout:

[ Person selector ]
[ Mood selector ]
[ Optional message ]

[ RANDOMIZE ]

[ BIG CTA: DEDICATE ]

---

## Generation State

NO spinner.

Use:
- waveform animation
- pulsing glow
- text feedback

---

## Output Screen

Order:
1. Poster (hero)
2. Track 1
3. Track 2
4. Track 3

---

## Audio Player

- Disc style player
- Subtle rotation
- Glow on play

---

## Microinteractions

Level 2:

- Hover → scale
- Click → compress
- Cards → tilt
- Transitions → smooth

---

## Motion System

- No static screens
- Always subtle movement

Types:
- float
- pulse
- glide

---

## Feedback UI

- Like/dislike buttons
- Simple, non-intrusive

---

## Share UI

- Beautiful preview card
- Name + mood + poster
- Clean CTA

---

## Mobile UX

- Full screen flows
- Bottom sheets for actions
- Thumb-friendly layout

---

## Sound Design

- Completion jingle (subtle)
- Button clicks (light)

---

## Final Rule

If it feels like:
- Stripe → wrong
- Notion → wrong

If it feels like:
- a creative playground → correct
