# Dhun — Context

## Product Definition

Dhun is an AI-powered music dedication platform that allows users to create personalized songs, posters, and lyrics instantly and share them.

Primary identity:
- Not a tool
- Not a generator
- A **digital emotional artifact engine**

Positioning:
- ChatGPT for music
- Canva for music
- Gift engine for the internet

---

## Core Product Split

### 1. Dedication (Primary)
- Fast, emotional, transactional
- Focus on shareability + monetization

### 2. Studio (Secondary)
- Power-user layer
- Prompt + tag + regeneration system

---

## Core User Flow

User selects:
- Person
- Mood
- Optional message

System does:
- Gemini → builds prompt
- ACE → generates 3 tracks
- Nano Banana → generates poster
- Lyrics generated

Output:
- Poster first
- Then 3 tracks (progressive)
- Lyrics on interaction

---

## Experience Flow

1. Click "Dedicate"
2. Cinematic transition
3. Poster appears
4. Waveform animation + anticipation
5. Tracks unlock progressively
6. User plays preview (10 sec free)
7. Monetization prompts appear

---

## Monetization Model

Currency: Coins (1 coin = ₹1)

- Generate (3 tracks): 6 coins
- Share full: 10 coins
- Download: 19 coins
- Full pack: 29 coins

Free:
- 20 coins onboarding
- Full quality preview (10 sec)

---

## Share System

Two layers:

### Free

### Paid
